Adaptive parsing test cases

  • College course assignment list Web pages. All HTML.
    • cs81 (HMC CS 81)
    • stems (HMC ENGR 59)
      • stems-trimmed (markup preceding and following the content removed)
  • Other HTML lists and tables.
    • cs-courses (list of Harvey Mudd College Department of Computer Science courses)
    • keunning (Geoff Keunning's list of publications)
    • wcstaff (list of Harvey Mudd College Writing Center staff)
      • wcstaff-trimmed (markup preceding and following the content removed)
    • wikihistory (edit history of a Wikipedia page)
  • Command output. Various formats that are trivially parseable with conventional methods.
    • dig
    • ls
    • dtrace
  • Other files.
    • sexp (data from stems formatted as an S-expression)
    • prefs-plist (a Mac OS X XML Property List)
    • citations (a BibTeX bibliography)
    • fenotes (MATLAB arrays representing the notes to "Fur Elise", from Cha/Molinder)
    • synthstems (data from stems formatted as XML; an early test used to develop the first version of the parser)
    • synthstems-noise (the synthstems file with two delimiter characters changed to the letter X)
